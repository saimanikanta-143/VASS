Ludo Rules & Regulations

Basic Rules

  • Objective: Move all four tokens from the starting area to the home area before opponents.
  • Players: Ludo is played by 2-4 players.
  • Movement: Players roll a die to determine how many steps they can move their tokens.
  • Starting Rule: A player must roll a six to move a token out of the starting area.
  • Capturing: If a player's token lands on an opponent's token, the opponent's token returns to the start.
  • Safe Zones: Some spots on the board are safe zones where tokens cannot be captured.

Common Violations

  • Moving a token without rolling the dice.
  • Not following the correct path on the board.
  • Rolling the dice out of turn.

Ludo Board Dimensions

Component Measurement
Board Size Typically 30 cm x 30 cm
Token Size Approximately 1.5 cm in diameter
Dice Size 1.6 cm per side
Number of Squares 52 main path squares + 4 home paths per player
